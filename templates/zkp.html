The Math behind the Magic:
1. Verifier (Network): Generates a random challenge C.
2. Prover (Voter): Signs the challenge locally using modular exponentiation: S = C^d mod n.
3. Verifier (Network): Receives S and computes C' = S^e mod n using the Public Key.
4. If C == C', the voter is mathematically proven to own the Private Key, while the Private Key d remains completely hidden.
